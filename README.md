# NotebookLM Suggestions Toggle

This Chrome extension adds a button to NotebookLM to toggle the visibility of the suggestion chips (follow-up questions).

## Installation

1.  Clone or download this repository.
2.  Open Chrome and go to `chrome://extensions/`.
3.  Enable "Developer mode" in the top right corner.
4.  Click "Load unpacked".
5.  Select the `notebooklm-hide-suggestions` folder.

## Usage

A "Hide" / "Show" button will appear next to the share button in the NotebookLM interface. Clicking it will toggle the visibility of:
*   The suggestion bar (follow-up questions)
*   The source panel (left side)
*   The studio panel (right side)
